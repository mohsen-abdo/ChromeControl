## ğŸ‡ºğŸ‡¸ Description

**Chrome Control v3.0** is a comprehensive Windows utility developed by **Mohsen Abdo** to help digital marketers and power users manage multiple Google Chrome profiles efficiently. It provides advanced automation for opening profiles, deep cleaning browser data, and performing secure backups.

### âœ¨ Key Features

#### 1. Profile Management & Automation
*   **Bulk Opener:** Open multiple selected Chrome profiles with a single click.
*   **Smart Delay:** Set a custom delay (in seconds) between opening browsers to prevent system freezing.
*   **Auto-Navigate:** Automatically open a specific URL in all launched profiles.
*   **Manual Selection:** Ability to type/paste profile names manually.
*   **Control:** Pause, Stop, or Resume the operation at any time.
*   **Kill Switch:** Instantly close all running Chrome windows.

#### 2. Deep Cleaning Tool
Select specific profiles and clear exactly what you need:
*   Browsing & Download History.
*   Cookies & Site Data.
*   Cached Images & Files.
*   Passwords & Sign-in Data.
*   Autofill Form Data & Site Settings.
*   Hosted App Data.

#### 3. Backup & Restore Tool
Secure your data with three backup modes:
*   **Profile Files:** Backup user data files.
*   **Passwords:** Export passwords to CSV format.
*   **Registry Keys:** Backup Chrome-related registry entries.
